# Website DogsWorld

## Instalation

Run 'npm install' to install all the dependencies.

## Getting started

Run 'npm start' to launch webpack-dev-server

Run 'npm run dev' to build dev version of this app.

Run 'npm run prod' to build prod version of this app.

## Technologies Used

-   JavaScript ES6+
-   React
-   React-Router
-   Sass & BEM
-   Bootstrap
-   Webpack
-   Babel
